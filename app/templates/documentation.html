Начальное окно

При открытии приложения оно состоит из 3-х частей. Слева находится меню работы с графом, по центру рабочая область, а справа ведется история действий.


Рабочая область

В рабочей области отображается текущий граф.


Узлы

Двойным кликом по области создается новая вершина. Двойной клик на вершину удаляет её с графа. Вместе с вершиной удаляются все связанные с ней ребра.

Вершины графа можно перемещать по рабочей области. При включенной опции физики, вместе с вершиной перемещается весь граф.


Рёбра

По очередное нажатие на вершины создает между ними ребро. При последовательном, но не двойном клике по вершине, создается ребро-петля. Двойной клик на ребро удаляет его с графа.


Меню работы с графом

Меню работы с графом состоит из 4-х вкладок.

Каждая из этих вкладок отвечает за свою часть работы с графом.


Вкладка узлы

Вкладка «Узлы» дает возможность редактировать вершины графа. Для этого надо выделить необходимую вершины мышкой или ввести её id в поиск. Далее её данные переносятся в поля, где их можно редактировать. Нажав на кнопку изменить, данные вершины отредактируется на новые. Поле id нельзя изменить, при поиске в поле id имеет смысл писать только числа.


Вкладка рёбра

Вкладка «Рёбра» работает по схожему принципу, но соответственно с рёбрами графа.


Вкладка граф

Вкалдка «Граф» дает возможность сохранить граф под введенным именем, открывать и удалять сохраненные графы. Также есть функция очистки рабочей области, это происходит по средству нажатия на кнопку "Очистить". В поле имени можно ввести только кириллицу, латиницу и числа.


Вкладка настройки

Во вкладке «Настройки» можно включать и выключать физическое взаимодействие вершин. Этот параметр влияет на поведение вершин при их пермещении и сталкивании, а также устремляет их к центру рабочей области.

Также, есть возможность изменить начальные цвета вершин и рёбер при создании. По умолчанию они случайные.


История

История помогает отследить действия проводимые во время сеанса над графом.

В поле истории пишится лог проводимых операций над графом. Это как добавление, поиск, редактирование и удаление вершин и рёбер, так и сохранение, открытие и удаление графа.

История не сохраняется вместе с графом, а присутствует только во время работы. По нажатию на кнопку "Очистить" история удаляется.
